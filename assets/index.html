CSS Snippet Cheatsheet

This is what we will use to remember our CSS snippets. Please refer to and interact with the grid below to learn more!

Flexbox Row

Use these properties to create a Flexbox Row layout 

            .row {
            display: flex;
            flex-direction: row;
            flex-wrap: wrap;
        }

Flexbox Column

Use this to create a flexbox column layout 

            .column {
            display: flex;
            flex-direction: column;
        }

CSS Grid Layout

Build a 12 column grid using CSS grid 

            .grid {
            display: grid;
            width: 100%;
            grid-template-columns:
        repeat(12, 1fr);
        }

Linear Gradients

This will create a background of linear gradient 

            .linear-gradient-background {
            background-image: linear-gradient (
            rgba(232, 102, 236, 0.3) 0%,
            rgba(232, 102, 236, 0.6) 100%
            )
        }

Box Transition Glow

This will make your boxes glow when you hover them 

            .code-card .code-header {
            border-radius: 8px;
            transition: a;; 0.5s ease-in-out;

            .code-card:hover,
            .code-card:hover .card-header {
                box-shadow: inset 0px 0px 8px
                rgba(232, 102, 236, 1), 0 0 15px
                rgba(232, 102, 236, 1);
            }
        }

Overlay Card with Title

Use position properties and negative margins to raise elements higher than natural starting point 

.card-header {
            position: relative;
            margin-top: -20px;
        }
